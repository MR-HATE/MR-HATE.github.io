# 简介

  • 常用说法是 CDM(Code Division Multiplexing) 或 CDMA(Code Division Multiple Access)。
  • 各用户使用经过特殊挑选的正交码型，通信各方互不干扰。
  • 因为其良好的抗干扰性和隐蔽性（频谱类似白噪声，不易被敌人发现），最初用于军事通信。
  • 多个用户可以在同一时间使用相同的频带进行通讯。

要实现 CDMA ，必须解决以下三个关键问题：

  1. 要达到复用的目的就要有足够多的地址码，而这些地址码又要有良好的自相关的特性和互相关的特性。地址码产生技术是 CDMA 的基础。
  2. 在系统的接收端产生的本地地址码（简称为本地码）不但在码型的结构上应与对方发来的地址码一致，而且在相位上也要完全同步，只有用这种完全同步的本地码对 i 收到的信号进行相关检测，才能正确地提取能用的信号。地址码同步技术是 CDMA 最主要的环节。
  3. 系统内所有用户使用同一频率的载波，各用户可以同时接收或发送信号。为了个用户之间的相互干扰降到最低程度，并且使各个用户的信号占用相同的带宽， CDMA 必须与扩展频谱（简称扩频）技术相结合。

# 原理

# 地址码的指派

每个用户（站）在发送数据之前会被指派一个唯一的 mmmbit 码片序列，每个序列长度恰好等于 TTT 。这个码片序列就是前文说的地址码。（解决第一个问题）。

# 码片序列的特点

  1. 不同站点分配的码片序列各不相同。
  2. 两个不同站点分配的码片序列相互正交。
  3. 两个不同站点间的码片序列的反码也相互正交
  4. 任何一个码片向量和该码片向量自己的规格化内积为 111 。
  5. 任何一个码片向量和该码片向量自己的反码的规格化内积为 −1-1−1 。

# 发送数据

站点发送数据时，使用对方的码片序列调制要发送的信息。例如要发送的信息是比特 1 ，则发送他自己的 mmm 位 码片序列。如果发送的信息是比特 0 ，则发送该码片序列的二进制反码序列。

e.g.e.g.e.g.

S 站被指派的 8 位码片序列是 0 0 0 1 1 0 1 1 。当 S 发送比特 1 时，它就发送序列 0 0 0 1 1 0 1 1 ；而当 S 发送比特 0 时，它就发送序列 1 1 1 0 0 1 0 0 。

可以看出，假设 S 站要发送的信息的数据率为 bbb b/s 。由于每个比特要转换为 mmm 个比特的码片， 因此 S 站实际上发送的数据率提高到 mbmbmb b/s，同时 S 站可利用的宽度也提高到原来数据的 mmm 倍。

# 接收数据

接收端接收数据后，计算自己的码片序列与收到信号的内积。根据结果（内积是否为 0）判断接收到的信息是不是发送给自己的。由于任意两个站点的序列都正交，所以虽然各站点用同一频率通信，他们之间也不会相互干扰。这种通信方式就是扩频通信的一种，扩频信号为:

# 小结

站点 A 向站点 B 发送信息，A 就用 B 站点的码片序列调制所发送的信息。B 在接收时，就计算自己的码片序列与收到的信号的内积，虽然 B 站同时也会收到其他各站点的信息，但只有 A 站发来的信息内积结果不为 0 , 而与其他站发来的信息的内积计算结果都为 0 ，所以最终 B 只收到了 A 站发来的信息。

# CDMA 的特点

CDMA 有着鲜明的特点：

  1. 可以节省带宽资源，所有用户使用同一频率，占用相同的带宽；
  2. 减少干扰对通信的影响，可提高通信的话音质量和数据传输的可靠性；
  3. 增大通信系统的容量 (是 GSM 的 4~5 倍);
  4. 降低手机的平均发射功率；
  5. 保密性好；
  6. 灵活机动。
