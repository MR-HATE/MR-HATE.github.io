# 基本的文字强调（斜体与加粗）

在要强调的文字前加 * 来对文字进行加粗与强调。

  • *x* 表示把 x 渲染成斜体。
  • **x** 表示把 x 加粗。
  • ***x*** 表示把 x 加粗并渲染成斜体。

e.g.e.g.e.g.

师爷，高

县长，硬

黄老爷又高又硬
