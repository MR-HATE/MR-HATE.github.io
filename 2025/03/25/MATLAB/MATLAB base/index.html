# （2）矩阵赋值

语法：

“变量名” = [……] ;

MATLAB 里只有一个工作区，即便有多个代码同时运行，他们共用同一个工作区。因此可能出现错误结果，因此我们每次运行程序都要清空工作区。

# 注 2：在 MATLAB 编辑器中输入块注释

MATLAB 支持通过 %{ 和 %} 定义多行注释块。需注意以下语法规则：

  1. 符号必须单独成行： %{ 和 %} 必须分别位于独立行，且前后不能有其他字符（包括空格）。
  2. 不支持嵌套：块注释不能嵌套使用。
